HTML5のheadタグらへん(meta etc...)

html文書を書くときのメタ情報等々の決まりごと、書き方が曖昧なままでったので、まとめておきます。

html4、5がごちゃごちゃになっているので、整理も込めてhtml5のお作法だけ記載します

DOCTYPE html

<!DOCTYPE html>

いきなりmetaタグじゃないけれど、まずはこれってことで。この文書がHTML5で書かれてますよっていう宣言。html4のときはやたら長かったというのに、、、簡単になりました。

言語

<html lang="ja">

日本語で書いてます！宣言。

文字コード

<meta charset="utf-8"/>

前はcontent-typeとか書かなくちゃいけなかったのに、、、（略

Script

<script src="hoge.js"></script>

Javascriptの記述をタグ内に書くか、外部ファイルをsrc要素で指定します。<script>はデフォルトで type="text/javascript" を含んでいるので、シンプルです。

link

<link href="hoge.css" rel="stylesheet">

外部ファイルを参照するときに使用します。hrefで場所、relで参照ファイルの形式を必ず指定します。

author, keywords, description

<meta name="author" content="">
<meta name="keywords" content="">
<meta name="description" content="">

それぞれ、誰が作ったか、ページのキーワード、ページの説明を記載します。

viewport

スマートフォン向け機能でhtml文書の表示範囲を指定できます。

<meta name="viewport"　content="width=320, initial-scale=1, minimum-scale=0.5, maximum-scale=2">

OGP （Open Graph Protocol)

metaタグに情報を追加することでSNSサイトにその情報を渡すことができるらしい。これはまたの機会に調べるとして例を書いておく

  • タイトル

    <meta property="og:title" content="hoge!hoge!" />

  • 説明文

    <meta property="og:description" content="hoge~" />

  • サムネイル

    <meta property="og:image" content="hoge.jpg" />
