SAStrutsでサクサク開発(1) - 環境構築

Seaser

なんでもSAStrutsとEclipse + pluginが超スゲェらしいので、黙って公式ページにのとってIDEをPluginを導入します。

チュートリアルのセットアップ(公式ページ)

Eclipse

Eclipse Luna SR2 Packages > Eclipse IDE for Java EE Developers > 任意のOS(今回はMac64bit)

インストール

起動後、Workspace Launcher

Tomcat Launcher

Eclipseから

help > Install New Software > Work with > add

  • Name:
    • Seasar Plugin3.2
  • Location:
    • http://eclipse.seasar.org/updates/3.2/

Sysdeo/SQLI Tomcat Launcherをインストールします。

なんかEclipseからインスコできなかった

Works with Eclipse 4.4のものをDLします

Sysdeo Eclipse Tomcat Launcher plugin > tomcatPluginV331.zip

解凍して出てきたjarファイルを、/Applications/eclipse/plugins/ に置きます。

SAStrutsPlugin, ResourceSynchronizer

help > Install New Software > Work with > add

  • Name:
    • Seasar Plugin3.3
  • Location:
    • http://eclipse.seasar.org/updates/3.3/

なんか出てきても、すすむ。

インストールが完了するとアプリケーションの再起動を促されますので、迷わずYes。

プロパティエディタPlugin

help > Install New Software > Work with > add

  • Name:
    • PropertiesEditor
  • Location:
    • http://propedit.sourceforge.jp/eclipse/updates/

インストールが完了するとアプリケーションの再起動を促されますので、迷わずYes。

チュートリアル用ファイル

DL

Seaserダウンロード > 最新版 > Super Agile Strutsチュートリアル

DLしたものを解凍すると、”sa-struts-tutorial”フォルダが出てくるので、これをEclipseのWorkspaceに置きます。

Import

これでチュートリアルの準備はできました。
